페이지의 목적은 매장 스케줄 관리표
직접 로그인해서 수정/확인이 직급별로 구분되길 바람
1월-12월 개별로 사용하길 바람

관리자(대표자)는 사용자의 권한 및 가입상태를 변경 할 수 있어야 함

각 사용자는 자신의 스케쥴을 월별로 등록/수정 할 수 있음.
관리자(대표자)는 이를 최종 승인/수정/삭제 권한을 가짐.

관리자(대표자)가 매장의 이름을 결정하면, 해당 매장의 각 월 일자별, 사용자별 근무 형태를 입력할 수 있어야 함.
매장은 여러개일 수 있음.
사용자는 여러명일 수 있음.
사용자는 관리자가 생성한 매장 중 자신의 근무 매장에 자신의 데이터를 입력할 수 있음.
근무의 형태는 오픈/미들/마감/종일로 나뉨.
휴일의 경우 연차/반차/대체휴/병가/요청의 상태가 필요.

요일, 근무형태, 휴무일 등은 구분이 쉽도록 별도의 배경색으로 구분이 되어야 함.

전체 페이지 구성은 메인페이지, 마이페이지, 매장페이지, 관리자 페이지.

메인페이지에서는 로그인, 가입.
로그인 성공 시 마이페이지에서 매장을 선택, 관리자의 승인이 있으면 해당 매장 페이지로 이동 가능.
매장 페이지에서는 상단의 기능 이용 가능.
관리자 페이지는 관리자 로그인 시, 마이페이지를 대체.
관리자 페이지에서는 회원관리, 매장관리, 근무 및 휴일 승인, 변경, 관리등을 할 수 있음.
